1. 使用node.js下的 Express搭建的服务端demo
2. 创建工程目录
3. cmd转到该目录下，使用npm init命令创建 package.json文件
4. 使用命令npm install express --save   安装express依赖包
5. 使用命令一下命令安装相关依赖包 
npm install body-parser --save
npm install cookie-parser --save
npm install multer --save

6. 创建myserver.js，即服务的具体实现
7. 使用 node myserver.js命令启动服务
